      Description:
​This is my first project as a Computer Science student! It's a simple desktop calculator application built using C# and Windows Forms. I created this project to practice UI design and basic logic implementation in Visual Studio.
​       Features:
​Basic Arithmetic Operations: Addition, Subtraction, Multiplication, and Division.
​Clear (C) button to reset the calculator.
​User-friendly interface.
​Error handling (e.g., division by zero).
    Technologies Used:
​Language: C#
​Framework: .NET Framework / .NET Core
​IDE: Visual Studio 2022
​UI: Windows Forms (WinForms)
